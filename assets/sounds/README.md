# Ambient Sounds for RNK Free MapGen

Place your sound files here to enable ambient audio in generated dungeons.
The module expects the following filenames (in .mp3 format):

- dungeon-ambience.mp3
- cave-dripping.mp3
- howling-wind.mp3
- crackling-fire.mp3
- flowing-water.mp3
- eerie-chanting.mp3
- machinery.mp3
- forest.mp3
- underwater.mp3

You can change these mappings in `scripts/rnk-dungeon.js`.
