# Props: Frozen

Place PNG assets in this folder. Filename = prop key used by the theme system.

| File | Description |
|---|---|
| `ice-pillar.png` | Thick column of solid ice |
| `frost-crystal.png` | Frost crystal growth |
| `frozen-body.png` | Creature or person frozen in ice |
| `snowdrift.png` | Piled snowdrift |
| `icicle-cluster.png` | Hanging icicle cluster |
| `ice-wall.png` | Section of ice-covered wall |
| `frost-rune.png` | Rune carved in ice |
| `frozen-chest.png` | Chest encased in ice |
| `blizzard-shrine.png` | Shrine to a cold deity |
| `cracked-ice.png` | Fractured ice sheet on floor |
