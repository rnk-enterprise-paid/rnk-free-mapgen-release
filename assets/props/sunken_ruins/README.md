# Props: Sunken Ruins

Place PNG assets in this folder. Filename = prop key used by the theme system.

| File | Description |
|---|---|
| `broken-column.png` | Toppled or cracked stone column |
| `seaweed-ruin.png` | Seaweed draped over stonework |
| `barnacle-wall.png` | Barnacle-encrusted wall section |
| `sunken-chest.png` | Chest half-buried in silt |
| `ancient-idol.png` | Barnacled ancient deity idol |
| `crumbled-arch.png` | Partially collapsed archway |
| `silt-pile.png` | Silt and sediment pile |
| `old-anchor.png` | Rusted ship anchor |
| `coral-growth.png` | Coral growing over ruins |
| `mosaic-remains.png` | Faded mosaic floor remnant |
