# Props: Forest

Place PNG assets in this folder. Filename = prop key used by the theme system.

| File | Description |
|---|---|
| `tree-stump.png` | Large hewn tree stump |
| `roots.png` | Gnarled exposed roots |
| `fern.png` | Fern cluster ground cover |
| `vine.png` | Hanging or creeping vine |
| `fallen-log.png` | Mossy fallen log |
| `mushroom-forest.png` | Cluster of forest mushrooms |
| `bush.png` | Dense bush / thicket |
| `hollow-tree.png` | Hollow dead tree trunk |
| `druid-stone.png` | Moss-covered standing stone |
| `animal-den.png` | Dug-out animal den burrow |
