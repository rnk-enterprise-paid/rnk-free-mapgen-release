# Props: Abandoned Mine

Place PNG assets in this folder. Filename = prop key used by the theme system.

| File | Description |
|---|---|
| `support-beam.png` | Wooden support beam / pit prop |
| `mine-cart.png` | Rusted mine cart on rails |
| `pickaxe.png` | Abandoned pickaxe or tool |
| `ore-vein.png` | Exposed ore vein in wall/floor |
| `lantern-mine.png` | Hanging miner's lantern |
| `collapsed-shaft.png` | Rubble from a cave-in |
| `pulley.png` | Rope-and-pulley winch system |
| `dynamite.png` | Bundle of old dynamite/blasting sticks |
| `tracks.png` | Section of mine cart track |
| `crate-mine.png` | Weathered supply crate |
