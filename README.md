# RNK Free MapGen

Foundry VTT module for dungeon map generation. Requires the RNK map server running locally or on your network.

Compatible with Foundry VTT v13 and v14.

## Screenshots

![RNK Free MapGen](RNK%20Free%20Mapgen.jpeg)

## v1.0.3

**Dungeon Generation**

Select from 21 dungeon themes: Abandoned Mine, Arcane Laboratory, Celestial, Classic Dungeon, Clockwork, Corrupted Temple, Crypt, Crystal Cavern, Drow Temple, Eldritch, Forest, Frozen, Fungal Underdark, Infernal, Lava, Modern, Natural Cave, Sewer, Sunken Ruins, Underwater, and Volcanic. The selected theme applies matching floor textures to the generated scene.

Map width and height are set independently from 20 to 200 tiles. Grid size, tile size (8, 12, or 16px), and grid type (square or hex) are configured before generation. A fixed seed can be entered for repeatable results. Scene name can be set manually or left for auto-generation.

Generated maps display in the GM Hub preview panel. Export to PNG, SVG, or JSON. Free tier allows 3 generations per day; the counter resets in 24 hours. Additional daily generations unlock through Patreon.

**Patreon Integration**

Connect a Patreon account from the GM Hub header to increase the daily generation limit.
