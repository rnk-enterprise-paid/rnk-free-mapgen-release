# RNK Free MapGen Release Shell

This repository is a lightweight distribution shell for the free tier. It should stay small: release manifest, release zip, and this README.

## What ships
- `module.json`
- `module.zip`
- Tier-specific release notes

## Tier indicators
- `[INCLUDED]` Theme-driven dungeon textures
- `[INCLUDED]` Compendium mob spawning
- `[INCLUDED]` Loot spawning and death drops
- `[INCLUDED]` Campaign mode support
- `[INCLUDED]` Scene-warp stair links
- `[INCLUDED]` Patreon login requirement
- `[LOCKED]` Expanded trap and architecture controls
- `[LOCKED]` Broader dungeon layout control surface

## Access model
- Free tier users get the base generation flow.
- Authentication is still required.
- Daily usage is limited by the API.

## Install path
Point Foundry to the manifest URL for this tier’s release repo.
